=====================================
ANIMATION PATTERNS
=====================================

Entry Animation (common):
  opacity: 0 -> 1
  transform: y: 20px -> 0

Hover Animation:
  scale: 1 -> 1.05
  opacity: 100% -> 80%

Continuous Animation:
  rotate: 0 -> 360° (loop)

Scroll Trigger:
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true }}

CUSTOMIZATION QUICK LINKS

  → components/sections/ProductGrid.tsx (line ~12)
  → Edit products array

Update Blog Posts:
  → components/sections/BlogSection.tsx (line ~8)
  → Edit blogPosts array

  → components/Header.tsx (line ~30-40)
